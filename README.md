# 22i-0908_22i-1007 Project B

Final submission workspace for the Web Programming project.

## Project Structure

- `frontend/` - React + Vite client application.
- `backend/` - Express + MongoDB/Mongoose API server.
- `docs/` - Rubric and project documentation.

## Incremental Module Plan

1. Base project import and clean folder structure.
2. Authentication and session management.
3. Password security and reset flow.
4. Role-based access control for admin, client, and freelancer.
5. Core marketplace features: gigs, orders, messages, reviews, transactions.
6. Form validation and error handling.
7. UI/UX polish, responsive layout, footer/navigation consistency.
8. Documentation, demo notes, and final submission cleanup.

## Local Setup Overview

Install dependencies separately in `frontend/` and `backend/`.

Backend environment values should be created from `backend/.env.example`.
